tapshex
=======

Generate ShEx from tabular application profiles in DCTAP format.

Installation
------------

.. code-block:: bash

    $ git clone https://github.com/tombaker/tapshex.git
    $ cd tapshex
    $ python -m venv .venv
    $ source .venv/bin/activate
    $ python3 -m pip install flit Pygments
    $ flit install -s
